# CDK Commands

## CDK Init App

```bash
cdk init app --language typescript [preferred_language]
```

## CDK Bootstrap

```bash
cdk bootstrap --profile [aws_profile]
```

## Generate CDK out config template

```bash
cdk synth (generates the cdk.out config template)
```

## CDK Deploy

```bash
cdk deploy --profile [aws_profile]
```
